# Hidden Word Duel

A real-time 1v1 word guessing game.

## Quick Start

1. Install dependencies:
   - `cd client && npm install`
   - `cd ../server && npm install`
2. Run backend:
   - `cd server && npm run start:dev`
3. Run frontend:
   - `cd client && npm run dev`
4. Open `http://localhost:3000`, enter a username, and find a match.

## Stack

- Frontend: Next.js + React + Socket.IO Client
- Backend: NestJS + Socket.IO + Prisma
